# NewBlog

个人新版博客，基于vue3构建，主要用于支持信息记录，chatgpt工具，家庭嵌入式设备操作

### 本地开发

该博客使用`pnpm`作为包管理工具，因此请先安装`pnpm`后在`new-blog`目录下执行以下指令

`pnpm i`

`pnpm run dev`
